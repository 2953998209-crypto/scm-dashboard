# 供应链食盐智能看板系统

> 智能化盐业供应链管理看板，集成库存进销存、客户台账、配送线路、销售考核于一体。

## 📊 功能模块

- **库存进销存看板** — 实时库存监控、进销存趋势分析、预警机制
- **客户资料台账** — 四级片区架构管理、客户分级画像
- **分区配送线路看板** — 配送路线可视化、效率分析
- **销售考核看板** — 业绩追踪、KPI 达成率、排名分析

## 🚀 在线访问

**网址：** https://2953998209-crypto.github.io/scm-dashboard/

## 💻 技术栈

- HTML5 + CSS3 (深色科技风)
- ECharts 5.4 — 数据可视化
- SheetJS (xlsx) — Excel 数据导入导出
- PWA — 可安装为桌面/移动应用

## 📦 安装使用

### 方式一：在线访问
直接打开 [GitHub Pages 链接](https://2953998209-crypto.github.io/scm-dashboard/) 即可使用。

### 方式二：本地运行
```bash
git clone https://github.com/2953998209-crypto/scm-dashboard.git
cd scm-dashboard
# 用浏览器打开 index.html 即可
```

### 方式三：安装为应用（PWA）
- Chrome/Edge 浏览器中点击地址栏右侧的安装图标
- 安装后可作为独立应用使用，支持离线运行
